# 中奖概率分析与调整方案

## 当前概率配置分析

### 现有配置（TOTAL_ODDS = 1000）
```javascript
const ODDS = {
    NO_PRIZE: 999,      // 99.9% 不中奖
    SMALL_PRIZE: 9,     // 0.9% 小奖
    NORMAL_PRIZE: 1,    // 0.1% 中奖
    BIG_PRIZE: 0        // 0% 大奖
};
```

### 当前中奖率计算
- **小奖概率**：9/1000 = **0.9%**
- **中奖概率**：1/1000 = **0.1%**
- **大奖概率**：0/1000 = **0%**
- **总中奖率**：10/1000 = **1%**

### 问题分析
1. **中奖率过低**：平均需要刮100张才能中1次奖
2. **大奖缺失**：用户无法体验到大奖的刺激感
3. **游戏体验差**：长期不中奖会降低用户兴趣

## 新概率配置方案

### 用户需求
- 刮5张能中1个小奖：小奖概率 ≈ 20%
- 大奖刮10张左右：大奖概率 ≈ 10%
- 保持游戏的刺激性和可玩性

### 调整后的配置
```javascript
const ODDS = {
    NO_PRIZE: 700,      // 70% 不中奖
    SMALL_PRIZE: 200,   // 20% 小奖（刮5张中1次）
    NORMAL_PRIZE: 80,   // 8% 中奖
    BIG_PRIZE: 20       // 2% 大奖（刮50张中1次）
};
const TOTAL_ODDS = 1000;
```

### 新中奖率计算
- **小奖概率**：200/1000 = **20%**（平均刮5张中1次）
- **中奖概率**：80/1000 = **8%**（平均刮12.5张中1次）
- **大奖概率**：20/1000 = **2%**（平均刮50张中1次）
- **总中奖率**：300/1000 = **30%**（平均刮3.3张中1次）

## 玩法说明文案

### 游戏规则
- 每张彩票包含45个刮刮乐格子
- 刮开格子即可查看是否中奖
- 中奖格子会显示"喜"或"囍"字样

### 中奖概率
- **小奖（30-60元）**：20%（平均刮5张中1次）
- **中奖（80-900元）**：8%（平均刮12.5张中1次）
- **大奖（1万-100万元）**：2%（平均刮50张中1次）
- **总中奖率**：30%

### 奖金说明
- 小奖：30元、40元、50元、60元
- 中奖：80元、100元、300元、600元、900元
- 大奖：1万元、20万元、100万元
- "囍"字奖金翻倍
