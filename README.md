# 手气王刮刮乐游戏

## 版本信息
- **当前版本**: v2.2.0
- **更新日期**: 2026-03-06

## 项目简介
一款模拟真实刮刮乐体验的HTML5手机端游戏，采用喜庆的红金配色风格，支持触摸刮开效果，内置付费系统和余额管理。比拼手气，看看谁是真正的手气王！

## 核心功能

### 游戏规则
- **游戏区域**: 9×5 网格，共 45 个格子
- **格子类型**:
  - **喜** 字：刮开后获得标注奖金
  - **囍** 字：刮开后获得标注奖金的两倍
  - **其他图案**：显示金额但不计入中奖
- **所有格子都显示金额**，但只有"喜"和"囍"才算真正中奖

### 付费系统
- **新用户赠送**: 首次进入赠送2元余额
- **游戏消耗**: 每次游戏消耗2元
- **余额支付**: 余额充足时消耗余额继续游戏
- **扫码支付**: 余额不足时需扫码支付2元
- **数据持久化**: 余额数据保存在本地 localStorage

### 操作方式
- 用手指在银色区域滑动刮开
- 支持跨格连续刮开
- 当格子刮开超过 50% 时自动揭晓

### 中奖概率设置
| 奖级 | 中奖概率 | 平均刮几张 |
|-----|---------|-----------|
| 小奖 | 10% | 10张 |
| 中奖 | 3% | 33.3张 |
| 大奖 | 0.3% | 333.3张 |
| **总中奖率** | **13.3%** | **7.5张** |

### 奖金设置
| 奖级 | 奖金范围 | 具体金额 |
|-----|---------|---------|
| 🎉 小奖 | 2-8元 | 2、3、5、8元 |
| 🎊 中奖 | 20-35元 | 20、22、25、30、35元 |
| 🏆 大奖 | 90-120元 | 90、100、120元 |

- 囍字（双倍）概率：15%

## 技术实现

### 前端技术
- **HTML5 Canvas**: 实现刮开效果
- **CSS3**: 响应式布局、动画效果
- **原生 JavaScript**: 游戏逻辑
- **localStorage**: 本地数据持久化

### 核心特性
- 完全适配手机端，一屏显示
- 支持触摸和鼠标操作
- 实时余额显示和更新
- 银色刮刮层带纹理效果
- 多弹窗交互（玩法说明、支付、余额）

## 使用方法

### 本地运行
1. 在项目目录下启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
2. 浏览器访问：`http://localhost:8000/喜相逢2.html`

### 手机端测试
1. 确保手机和电脑在同一局域网
2. 查看电脑 IP 地址
3. 手机浏览器访问：`http://[电脑IP]:8000/喜相逢2.html`
