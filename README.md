# 手气王刮刮乐游戏

## 版本信息
- **当前版本**: v3.1.0
- **更新日期**: 2026-03-06

## 项目简介
一款模拟真实刮刮乐体验的HTML5手机端游戏，采用喜庆的红金配色风格，支持触摸刮开效果。每人5次机会，中奖金额可兑换更多游戏次数！

## 核心功能

### 游戏规则
- **游戏区域**: 9×5 网格，共 45 个格子
- **格子类型**:
  - **喜** 字：刮开后获得标注奖金
  - **囍** 字：刮开后获得标注奖金的两倍
  - **其他图案**：显示金额但不计入中奖
- **所有格子都显示金额**，但只有"喜"和"囍"才算真正中奖

### 游戏次数系统
- **初始次数**: 每人只有 **5次** 游戏机会
- **中奖金额**: 自动存入余额
- **兑换规则**: 每 **10元** 余额可兑换 **1次** 游戏机会
- **余额不足**: 余额不足10元时无法继续游戏

### 操作方式
- 用手指在银色区域滑动刮开
- 支持跨格连续刮开
- 当格子刮开超过 50% 时自动揭晓

## 技术实现

### 前端技术
- **HTML5 Canvas**: 实现刮开效果
- **CSS3**: 响应式布局、动画效果
- **原生 JavaScript**: 游戏逻辑
- **localStorage**: 本地数据持久化

### 核心特性
- 完全适配手机端，一屏显示
- 支持触摸和鼠标操作
- 实时余额和次数显示
- 银色刮刮层带纹理效果
- 多弹窗交互（玩法说明、兑换、次数不足）

## 使用方法

### 本地运行
1. 在项目目录下启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
2. 浏览器访问：`http://localhost:8000/lucky.html`

### 手机端测试
1. 确保手机和电脑在同一局域网
2. 查看电脑 IP 地址
3. 手机浏览器访问：`http://[电脑IP]:8000/lucky.html`

## 功能按钮
- **再来一次**: 开始新游戏（消耗1次机会）
- **?按钮**: 查看玩法说明
- **长按标题**: 重置所有游戏数据
